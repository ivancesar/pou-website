# POU Website Proof of Concept

This repository is an AI coding learning project and proof of concept for a possible POU website.

The site is a static example built with HTML, CSS, and JavaScript. It explores how AI-assisted coding can be used to quickly design, structure, and iterate on a small informational website for POU.

## Purpose

- Practice working with AI coding tools in a real project folder.
- Demonstrate a proof-of-concept website structure for POU.
- Provide an example site that can be reviewed, discussed, and improved over time.

## Project Type

This is not a final production website. It is a learning exercise and example implementation intended to show what could be built, refined, or expanded later.

## Contents

- Static HTML pages for the main site sections.
- Shared styling in `styles.css`.
- Basic interactive behavior in `script.js`.
- POU logo asset in `assets/`.
- Functional notes in `Documentation/`.
